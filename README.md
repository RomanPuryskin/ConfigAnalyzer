# Config Analyzer

CLI утилита для анализа конфигурационных файлов веб-приложений (YAML / JSON) на наличие потенциально небезопасных настроек.

---

##  Возможности

Утилита проверяет конфигурации на типовые проблемы безопасности:

- включенный `debug` режим
- пароль в открытом виде
- использование `0.0.0.0` без ограничений
- отключенная TLS-проверка
- использование устаревших или небезопасных алгоритмов (например MD5)
- слишком широкие права доступа к файлам

---

## Поддерживаемые форматы

- JSON
- YAML

---

## Запуск

### 1. Сборка бинарника

## 1.1 Через Go

```bash
go build cmd/main.go
```
## 1.2 Через Make

```bash
make build
```
### 2. Запуск

## 2.1 Через Go

```bash
go build -o ./cmd/main.go
```
## 2.2 Через Make

```bash
make run
```
